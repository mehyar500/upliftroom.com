# Backend (Supabase)

This directory contains the Supabase configuration for the project.

## Getting Started

1. Install Supabase CLI: https://supabase.com/docs/guides/cli
2. Initialize Supabase (if not already):
   ```bash
   supabase init
   ```
3. Start local Supabase:
   ```bash
   supabase start
   ```

## Structure
- `supabase/config.toml`: Configuration for local Supabase instance.
